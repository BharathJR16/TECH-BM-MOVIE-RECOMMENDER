# 📌 Project Overview

TECH BM Movie Recommender is an AI-powered movie recommendation system built using:

- 🧠 Machine Learning
- 🎬 Content-Based Filtering
- ⚡ Cosine Similarity
- 🎨 Streamlit Interactive UI

The system recommends similar movies based on:

✅ Genres  
✅ Cast  
✅ Keywords  
✅ Director  
✅ Overview  

Users simply enter a movie name and get:

✨ Top 5 Similar Movies  
✨ Ratings  
✨ Genres  
✨ Duration  
✨ Interactive Movie Pages

## 🎯 Smart Movie Search

- Handles spelling mistakes
- Uses fuzzy matching

| User Input | Detected Movie |

| spider man | Spider-Man 3 |
| avatr | Avatar |
| batmn | Batman |
